## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Test in the browser (no `npm run build`)

1. Keep **`npm run dev`** running.
2. Open Chrome (or any browser) and walk through flows manually — hot reload will pick up code changes.
3. Optional quick HTTP check (still no build):

   ```bash
   npm run smoke:dev
   ```

   This hits main public URLs plus protected routes (expects redirect to login when logged out).

**Manual checklist (logged out):** Home → Jobs → Companies → Blog article → Trends → Login/Register.

**Manual checklist (seeker):** Dashboard → Applications → Saved jobs → Resume → Profile → Messages.

**Manual checklist (employer):** Dashboard → Jobs → Applicants (list + Kanban) → Company → Billing.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
